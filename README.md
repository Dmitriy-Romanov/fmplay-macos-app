# FMPLAY macOS

Версия: `1.0.1`

FMPLAY macOS - небольшое нативное приложение для macOS, написанное на Rust, для прослушивания радиостанций из каталога [`fmplay.ru`](https://fmplay.ru/).

Приложение загружает актуальный список станций с FMPLAY, позволяет быстро искать радио по названию, переключать качество потока `16 / 24 / 32 / 48` кбит/с, выбирать темную или светлую тему, добавлять станции в избранное и видеть текущий трек там, где станция отдает metadata.

Интерфейс сделан под повседневное прослушивание: слева список станций, справа компактная панель текущей станции с обложкой, управлением воспроизведением, избранным, настройками и громкостью. Окно можно свернуть до узкого режима только с плеером и развернуть обратно.

![FMPLAY macOS screenshot](fmplay%20screenshot.png)

## Возможности

- Каталог радиостанций FMPLAY загружается автоматически.
- Избранные станции поднимаются вверх списка и сохраняются между запусками.
- Последняя выбранная станция, качество потока, громкость, тема и размер окна сохраняются локально.
- Для станций с metadata отображается текущий трек.
- Можно собрать полноценный `.app` bundle с иконкой FMPLAY.

## Запуск

```sh
cargo run
```

Настройки приложения сохраняются в:

```text
~/Library/Application Support/FMPLAY Radio/config.json
```

## Сборка

```sh
cargo build --release
```

Готовый бинарник будет в `target/release/fmplay_macos_app`.

## macOS app

Чтобы собрать `.app` с иконкой FMPLAY:

```sh
scripts/build_app.sh
```

Готовое приложение будет в:

```text
target/release/FMPLAY Radio.app
```

### Если macOS блокирует приложение

После скачивания release с GitHub macOS может оставить на приложении quarantine attribute. Если `.app` не запускается на другом Mac, выполните:

```sh
xattr -rd com.apple.quarantine "FMPLAY Radio.app"
```

## GitHub Actions и релизы

В репозитории настроен GitHub Actions workflow:

- при push в `master` собирается macOS `.app` и загружается build artifact;
- при push тега вида `v1.0.1` дополнительно создается GitHub Release;
- в release прикладывается архив `FMPLAY Radio-macOS.zip`.

Чтобы выпустить релиз:

```sh
git tag v1.0.1
git push origin v1.0.1
```
